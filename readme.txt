**** Startinstruktioner ****

* Importera "\importdb.sql" till HeidiSQL
* Öppna "\Backend\" mappen i IntelliJ
* Ändra eventuellt lösenord och port i "\Backend\src\main\resources\application.properties"
* Starta backend Java server "\Backend\src\main\java\com\example\Holidaymaker\HolidayMakerApplication.java"
* Öppna "\Frontend\" mappen i Visual Studio
* Kör "npm i" i terminalen
* Starta Stripe server "\Frontend\server.js" i Visual Studio med hjälp av "npm start" i terminalen
* Starta Vue servern genom att köra "npm run serve" i terminalen
